$$
D \left( \sum _ { i = r } ^ { s } \alpha _ { i } N _ { i } ^ { m } ( x ) \right) = ( m - 1 ) \sum _ { i = r } ^ { s + 1 } \frac { \alpha _ { i } - \alpha _ { i - 1 } } { t _ { i + m - 1 } - t _ { i } } N _ { i } ^ { m - 1 } ( x )
$$
其中 $\alpha_{r-1}=\alpha_{s+1}=0$

$$

$$

$$
